{% blocktrans %} The key functions of the course site are as follows: {% endblocktrans %}

  • {% blocktrans %} in ① you can select “ACTIONS” in order to access the following functions: {% endblocktrans %}
    • {% blocktrans %} “ADD TO/ REMOVE FROM FAVOURITE”: Add/Remove the viewed course to your list of favourite courses. {% endblocktrans %}
    • {% blocktrans %} DUPLICATE COURSE: Duplicate this course {% endblocktrans %}
    • {% blocktrans %} EXPORT COURSE: Export all course contents in a PDF {% endblocktrans %}
    • {% blocktrans %} EDIT COURSE - more information in a later section {% endblocktrans %}
    • {% blocktrans %} EDIT STRUCTURE - more information in a later section {% endblocktrans %}
    • {% blocktrans %} VIEW HISTORY - more information in a later section {% endblocktrans %}
    • {% blocktrans %} DELETE COURSE: Delete the course {% endblocktrans %}
  • {% blocktrans %} in ② you can click on the name of a specific topic for quick topic jumping, and click on the hamburger button to toggle/untoggle the topic list. {% endblocktrans %}
  • {% blocktrans %} in the “Coursebook” part you can see marked coursebook content. In ③ you can also click on the hamburger button in order to toggle/untoggle the coursebook section. {% endblocktrans %}
  • {% blocktrans %} if there is any content in your “Coursebook”, there will be an “EXPORT” button (see ④), which you can select to download a pdf containing all coursebook content. {% endblocktrans %}
  • {% blocktrans %} on the right side of each topic there is an “ADD CONTENT” button (see ⑤). Click to add new content to the corresponding topic. {% endblocktrans %}
